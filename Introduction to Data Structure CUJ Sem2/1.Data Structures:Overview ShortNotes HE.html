1. Definition & Introduction

Definition: Data Structure basically data elements ke beech ka logical relationship hota hai. Simple shabdon mein kahein toh, data structure ek tarika hai data ko computer mein organize karne ka taaki hum us data ko efficiently use kar sakein.

Jab hum koi program banate hain, toh humein do cheezon ki zarurat hoti hai:

Program = Algorithm + Data Structure

Algorithm ek step-by-step procedure hai problem solve karne ka, aur Data Structure wo tarika hai jisse data organize kiya jata hai taaki algorithm sahi se kaam kare.

2. Classification of Data Structures

Data Structures ko mainly do broad categories mein divide kiya gaya hai:

Primitive Data Structures

Ye basic structures hote hain jo system/machine level instructions par directly operate karte hain.

  • Integer (Numbers)
  • Float (Decimal numbers)
  • Character (Single alphabet/symbol)
  • Pointer (Memory address holder)

Non-Primitive Data Structures

Ye sophisticated hote hain aur Primitive DS se derive kiye jate hain. Ye same type (homogeneous) ya alag type (heterogeneous) data ka group ho sakte hain.

1. Linear List: Arrays Linked List Stack Queue
2. Non-Linear List: Trees Graphs

Difference (Antar)

Primitive generally language mein built-in hote hain (int, float).
Non-Primitive primitive se milkar bante hain aur complex data relationships ko handle karte hain (Graph, Tree).

3. Arrays & Lists

Arrays

Array ek collection hai finite number of homogeneous elements ka (yani sabhi elements same type ke hone chahiye, jaise sirf integers).

  • Consecutive Memory: Array ke elements memory mein lagatar (line se) store hote hain.
  • Indexing: Access karne ke liye hum index use karte hain `arr[0]`. Index hamesha 0 se shuru hota hai.
  • Size Formula: Elements count = (Upperbound - Lowerbound) + 1
Example: Agar array index 0 se 9 tak hai, toh size = (9-0)+1 = 10 elements.

Linked Lists

Linked List data items ka collection hai jisme har element ko Node kehte hain. Har node ke paas do fields hoti hain:

  1. Information Field: Actual data store karta hai.
  2. Pointer Field: Next element ka address store karta hai.

Types:

Single Linked List Doubly Linked List Circular Linked List

4. Stack & Queue

Stack (LIFO)

Stack plates ke dher jaisa hota hai. Jo plate sabse last mein rakhi gayi, wo sabse pehle uthai jayegi. Isse LIFO (Last In First Out) kehte hain.

  • Insertion = PUSH
  • Deletion = POP
  • Insertion aur Deletion sirf ek end se hota hai jise TOP kehte hain.

Queue (FIFO)

Ye ticket counter ki line jaisa hai. Jo insaan pehle aaya, wo pehle jayega. Isse FIFO (First In First Out) kehte hain.

  • Insertion = Rear End se hota hai.
  • Deletion = Front End se hota hai.
  • Real world example: Railway reservation row.

5. Trees & Graphs

Trees

Tree ek hierarchical structure hai (jaise family tree). Isme sabse upar Root node hota hai. Data "branches" mein organize hota hai. Note: Computer Science mein Tree ulti grow karti hai (Root upar, branches neeche).

Graph

Graph ek mathematical structure hai jo physical structures ko represent karta hai. Definition: G(V,E) where V = Vertices (points) and E = Edges (lines connecting points).

Test Your Knowledge (10 Questions)

Short Answer Questions

Q1. Define Data Structure.

Data structure is a way of organizing all data items that considers not only the elements stored but also their relationship to each other. It affects the design of both structural and functional aspects of a program.

Q2. What is the equation for an Array's size?

Size = (Upperbound - Lowerbound) + 1.

Q3. Differentiate between Stack and Queue.

Stack follows LIFO (Last In First Out) principle where insertion/deletion happens at TOP. Queue follows FIFO (First In First Out) principle where insertion happens at REAR and deletion at FRONT.

Q4. What are Primitive Data Structures?

These are basic structures directly operated upon by machine instructions. Examples: Integer, Float, Character, Pointer.

Q5. Define Graph.

A graph G(V,E) is a non-linear data structure consisting of a set of vertices (V) and a set of edges (E) connecting them.
