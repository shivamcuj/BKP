Fundamentals of AI

Understanding the Task Environment

Before designing any AI agent, we must define its environment. We do this using the PEAS framework.

View Examples Learn Concepts

The PEAS Framework

P

Performance Measure

The criteria used to determine how successful an agent is.

"Did it work well?"
E

Environment

The external world or context where the agent operates.

"Where is it?"
A

Actuators

The tools or mechanisms the agent uses to perform actions (Output).

"How does it act?"
S

Sensors

The devices used to perceive data from the environment (Input).

"How does it see?"

Agent Examples

High Priority

Automated Taxi

Performance Safe, Fast, Legal, Comfortable trip, Maximize profits.
Environment Roads, other traffic, pedestrians, customers.
Actuators Steering, accelerator, brake, signal, horn, display.
Sensors Cameras, speedometer, engine sensors, GPS.

Medical Diagnosis System

Performance Healthy patient, reduced costs.
Environment Patient, hospital, staff.
Actuators Display of questions, tests, diagnoses, treatments, referrals.
Sensors Keyboard entry of symptoms, findings, patient answers.

Satellite Image Analysis

Performance Correct image categorization.
Environment Downlink from orbiting satellite.
Actuators Display of scene categorization.
Sensors Color pixel arrays.

Part-Picking Robot

Performance Percentage of parts in correct bins.
Environment Conveyor belt with parts, bins.
Actuators Jointed arm and hand.
Sensors Camera, joint angle sensors.

Refinery Controller

Performance Purity, yield, safety.
Environment Refinery, operators.
Actuators Valves, pumps, heaters, displays.
Sensors Temperature, pressure, chemical sensors.

Interactive English Tutor

Performance Student's score on test.
Environment Set of students, testing agency.
Actuators Display of exercises, suggestions, corrections.
Sensors Keyboard entry.

Exam Cheat Sheet

Core Definition

P: Success Metrics E: The World A: Outputs (Actions) S: Inputs (Perception)
Automated Taxi Tips
  • Sensors: Must include GPS & Cameras.
  • Actuators: Must include Horn & Signals (Communication).
  • Perf: Safety is priority #1, then Speed/Profit.
Medical System Tips
  • Sensors: Keyboard entry (Symptoms) is key.
  • Actuators: It outputs Referrals and Questions.
  • Environment: Includes Patients AND Hospital Staff.
Exam Strategy: When asked about PEAS, always draw a 4-column table. Start by defining the Environment first, as it dictates the Sensors and Actuators needed.
