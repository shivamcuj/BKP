AI Agents: The Building Blocks of AI

An AI agent is more than just code; it's a goal-oriented system that interacts with the world. Based on the Fundamentals of AI curriculum by Dr. Deepti Malhotra.

01

What is an AI Agent?

An AI agent is a software program or system designed to perceive its environment, make decisions, and take actions to achieve specific goals.

Common Examples:

Driverless cars, Siri Virtual Assistant, Chatbots.

Key Components

Sensors

For perception. Examples: Cameras, input data, microphones.

Actuators

For actions. Examples: Robot arms, text output, wheels.

Agent Function

The decision-making logic/mathematical mapping.

Environment

The world in which the agent operates.

03

Agent Structure

Agent = Architecture + Agent Program

Architecture

The physical machinery or device the agent executes on (e.g., PC, Robotic Car, Camera Hardware).

Agent Program

The software implementation of the agent function. It processes the percepts to produce actions.

04

Types of AI Agents

Simple Reflex Agents

Reactive

Acts only on the current perception. It ignores the rest of the history.

  • Based on If / Then Rules
  • Environment must be Fully Observable
  • Example: Thermostat.

Model-Based Reflex

Internal State

Maintains an internal state to track things it cannot see now.

  • Good for Partially Observable worlds.
  • Uses a "model" of how the world works.
  • Example: Self-driving car with a map.

Goal-Based Agents

Strategic

Expansion of model-based agents with a desirable situation (Goal).

  • Uses Searching and Planning
  • Evaluates "What if I do A?" against the goal.
  • Example: Chess-playing AI.

Learning Agents

Adaptive

Learns from experience and improves performance over time.

  • Acts, observes results, and updates logic.
  • Critical for dynamic environments.
  • Example: Recommendation system.
