Task Environment & PEAS

Kisi bhi AI Agent ko design karne se pehle uska Task Environment define karna zaroori hai. Iske liye hum PEAS formula use karte hain.

Performance Environment Actuators Sensors
P

Performance Measure

Ye decide karta hai ki agent kitna successful hai.

"Kaam kaisa kiya? Fast? Safe?"

E

Environment

Wo surroundings jahan agent operate karega.

"Road? Hospital? Factory?"

A

Actuators

Wo tools jinse agent actions leta hai (Output).

"Steering, Screen, Robotic Arm"

S

Sensors

Wo devices jinse agent data perceive (dekhta/sunta) hai.

"Camera, GPS, Keyboard"

PEAS Examples (From Notes)

Most Important

Automated Taxi

Performance: Safe, Fast, Legal drive, Maximize Profit, Comfortable trip.
Environment: Roads, Traffic, Pedestrians (log), Customers.
Actuators: Steering, Accelerator, Brake, Signal, Horn, Display.
Sensors: Cameras, Speedometer, GPS, Engine sensors.

Medical System

Performance: Patient ki health, cost kam karna.
Environment: Patient, Hospital, Staff.
Actuators: Display questions, Diagnoses, Treatments refer karna.
Sensors: Keyboard entry (symptoms), Patient answers.

Part-Picking Robot

Performance: % of parts in correct bins.
Environment: Conveyor belt, bins (dabbe).
Actuators: Jointed Arm (haath), Hand.
Sensors: Camera, Joint angle sensors.

English Tutor

Performance: Student ka test score.
Environment: Students, Testing agency.
Actuators: Display exercises, Corrections dikhana.
Sensors: Keyboard entry.

Exam Quick Revision

Definition

PEAS stands for Performance, Environment, Actuators, Sensors.

Performance Measure Success criteria (e.g., Safety, Speed, Score).
Environment The world (e.g., Road, Patient, Satellite Downlink).
Actuators (Outputs) Action tools (e.g., Steering, Display, Valves).
Sensors (Inputs) Perception tools (e.g., Camera, Keyboard, GPS).

"Exam mein agar 'Taxi Driver' ka example aaye, toh Actuators mein 'Steering/Brake' aur Sensors mein 'Camera/GPS' likhna mat bhoolna."
