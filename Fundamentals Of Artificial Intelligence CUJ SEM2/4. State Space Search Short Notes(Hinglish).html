State Space Search

AI mein problems solve karne ka tareeka. Jaise Google Maps raasta dhoondhta hai, waise hi AI State Space Search use karke solution tak pahunchta hai.

What is State Space Search?

Jab humein kisi complex problem ka solution chahiye hota hai (jaise Chess game jeetna ya Puzzle solve karna), toh AI sabhi possible situations ko explore karta hai.

Graph Structure

Is process ko ek Graph ya Tree ki tarah visualize kiya ja sakta hai:

  • Nodes: Problem ki alag-alag States (situations).
  • Edges: Ek state se dusri state jaane ka transition (Action).

The Math ($S$)

Formal definition: S: {S, A, Action, Transition, Cost}
Jahan S States hain aur A Actions hain.

Key Terms (Terminologies)

State

System ka ek snapshot. Kisi particular time pe problem kaisi dikhti hai.

Start & Goal State

Start: Jahan se hum shuru karte hain.
Goal: Jahan humein pahunchna hai (Manzil).

Operators / Actions

Wo steps jo hum le sakte hain ek state se dusri state mein jaane ke liye.

Path & Cost

Path: Start se Goal tak ka poora rasta.
Cost: Us raste mein kitni energy/time laga.

Classic Example: The 8-Puzzle

Ye AI ka sabse common example hai. Ek 3x3 board hota hai jisme 1 se 8 tak numbered tiles hoti hain aur ek Blank Space hoti hai.

Goal:

Tiles ko slide karke correct order (1-8) mein arrange karna.

Actions (Operators):

Blank space ko Move karna: UP, DOWN, LEFT, RIGHT.

Path Cost:

Har move ka cost = 1 step.

Real World Examples

Route Finding (Google Maps)

Imagine aapko Hostel se Exam Hall jaana hai.

  • Start: Current Location (Hostel).
  • Goal: Destination (Library/Hall).
  • Actions: Turn Left/Right, take Bus/Walk.
  • Search: Google maps best rasta dhoondhta hai.

Water Jug Problem

Ek classic AI puzzle.

  • Goal: Specific amount of water measure karna using 2 jugs.
  • Constraint: Jugs par marking nahi hoti.
  • Action: Fill jug, Empty jug, Pour one into another.
