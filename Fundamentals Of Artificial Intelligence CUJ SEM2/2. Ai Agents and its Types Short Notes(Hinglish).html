What is an AI Agent?

AI Agent basically ek software program hai jo apne environment ko perceive (samajhta) karta hai, decisions leta hai, aur phir action leta hai.

Hinglish Core:

Simple words mein: Ye ek "Smart Assistant" jaisa hai. Jaise Siri ya Self-driving cars. Ye sensors se environment ko dekhte hain aur actuators se respond karte hain.

Sensors

For Perception (Cameras, Data Input)

Actuators

For Actions (Robot arms, Text Output)

Agent Function

Decision-making Logic (Dimaag)

Structure of an Agent

Agent = Architecture + Agent Program

1. Architecture

Ye wo machinery hai jispe agent execute hota hai. Jaise ki aapka PC, Robotic car, ya cameras.

Hinglish Tip: Architecture matlab agent ka "Body".

2. Agent Program

Ye agent function ka implementation hai. Ye logic provide karta hai actions perform karne ke liye.

Hinglish Tip: Agent Program matlab uska "Brain/Software".

Types of AI Agents

Basic

Simple Reflex Agents

Ye agents sirf current perception ke basis pe act karte hain. Inhe past history se koi matlab nahi hota.

  • Based on If-Then Rules
  • Environment must be fully observable.
  • Ex: Thermostat (Auto AC on/off).
Advanced

Model-Based Reflex

Ye ek internal state maintain karte hain duniya ko samajhne ke liye. Ye partially observable environments mein kaam aate hain.

  • Uses map & sensor data.
  • Ex: Self-driving car navigation.
Strategic

Goal-Based Agents

Inka ek specific goal hota hai. Ye actions tabhi select karte hain jab goal achieve hone wala ho.

  • Uses Searching and Planning.
  • Ex: Chess-playing AI.
Adaptive

Learning Agents

Ye agents experience se seekhte hain aur apni performance ko time ke saath improve karte hain.

  • Improves with every mistake/success.
  • Ex: YouTube recommendations.
